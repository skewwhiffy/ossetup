# ossetup
Setup scripts for OSes

## Arch linux
* Boot up in EFI mode into the Arch ISO.
* `pacman -Sy git`
* `git clone https://github.com/skewwhiffy/ossetup`
* `cd ossetup/arch`
* `./format.sh`
* `./install.sh`
* You are now in your new system.
* `git clone https://github.com/skewwhiffy/ossetup`
* `cd ossetup/arch`
* `./setup.sh`
* You should now have a bootable system
